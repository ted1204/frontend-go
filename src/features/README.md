# Features Directory

Feature-based architecture for better code organization and maintainability.

## Structure

Each feature contains:

- `components/` - Feature-specific React components
- `pages/` - Feature route pages
- `services/` (optional) - Feature-specific API services
- `hooks/` (optional) - Feature-specific custom hooks

## Features

- **admin/** - Admin dashboard and management
- **auth/** - Authentication (signin, signup)
- **forms/** - Form submissions and management
- **groups/** - Group management and membership
- **projects/** - Project creation and management
- **storage/** - File storage and browser
- **monitoring/** - Real-time monitoring and logs
